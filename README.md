# spa-vux-quickstart

## How use

``` bash
# 由于众所周知的原因，可以考虑切换源
$ npm set registry https://registry.npm.taobao.org/

# 如果未安装
$ npm install -g vue-cli

# 开始
$ vue init yijian002/spa-vux-quickstart my-project
$ cd my-project
$ npm install
$ npm run dev
```

## What is VUX

VUX 是基于 WeUI 和 Vue.js 的 移动端 UI 组件库，提供丰富的组件满足移动端(微信)页面常用业务需求。
